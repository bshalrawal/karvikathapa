# Karvika Thapa – Premium Landing Page

Single-page responsive site with light/dark/system theme, animations, dynamic content, and top navigation + footer.

## Quick start
- Open `index.html` in a browser, or serve locally:
  - Python: `python3 -m http.server 8080` then visit http://localhost:8080/

## Edit content
- Update `content.json` with accurate details (name, bio, stats, work, publications, contact, footerLinks).
- No code changes required for content updates.

## Features
- System theme detection + manual toggle (persists)
- Intersection-based reveal animations, counters, parallax orbs
- Responsive navigation with mobile drawer
- Accessible colors and reduced motion support
